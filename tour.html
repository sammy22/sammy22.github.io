Tour Companion - IIIT Hyderabad 2017

To enhance the experience of tourism a mobile application was developed to make useful information available more readily and a web portal to simplify the process planning visits to tourist destinations. These solutions developed were tested for Nehru Zoological Park, Hyderabad. The mobile app and and web portal are generic in nature and can be used at any tourist spot as long as data has been collected for that spot. Data on app usage is collected which is used to improve the overall experience over time. The data generated for each user can be used to study the tourist spots like the points of interest, most visited sites ,time spent at each site and much more



Tech stack

  • OpenStreetMaps
  • NodeJS
  • Python
  • OpenCV
  • Scrapy
